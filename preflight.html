§0 · Before the lecture

Install & download
before we start.

~10 minutes of setup, done at home — so we don’t spend the workshop debugging Rust.

A · Toolchain

# 1. Rust (circom is written in Rust)
curl --proto '=https' --tlsv1.2 -sSf \
  https://sh.rustup.rs | sh

# 2. circom 2.x — clone & build
git clone https://github.com/iden3/circom.git
cd circom && cargo build --release
cargo install --path circom

# 3. snarkjs (proving / verifying)
npm install -g snarkjs

# sanity check
circom  --version
snarkjs --version
