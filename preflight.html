§0 · Before the lecture

Install & download
before we start.

~10 minutes of setup, done at home — so we don’t spend the workshop debugging Rust.

1

Install Rust

circom is written in Rust — you need the toolchain first.

rust-lang.org/tools/install ↗

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
2

Build & install circom

Clone the repo, compile with Cargo, and install the binary.

iden3/circom ↗  ·  docs.circom.io ↗ 

git clone https://github.com/iden3/circom.git
cd circom && cargo build --release
cargo install --path circom
3

Install snarkjs

The proving & verification toolkit — runs in Node.

iden3/snarkjs ↗ 

npm install -g snarkjs
4

Download the Powers of Tau file

A trusted-setup ceremony output. Reused across all circuits — download once. 2^20 supports up to ~1M constraints; 2^14 works for small demos.

wget https://storage.googleapis.com/zkevm/ptau/powersOfTau28_hez_final_20.ptau

ls -lh powersOfTau28_hez_final_20.ptau
# → 290 MB  ✔

Need a different size? See the full list of available ptau files → snarkjs README §7 ↗

✓

Sanity check

circom  --version
snarkjs --version
